📖 Instrucciones de uso
🚀
Primeros pasos
Cargar datos y crear tu primer panel
▶
  1. Cargar un fichero Excel — Pulsa 📂 Cargar datos en la barra superior o arrastra el fichero sobre la aplicación. Formatos admitidos: .xlsx, .xls y .csv. El fichero se procesa en memoria local, nunca se sube a ningún servidor.
  2. Ir a Configuración — Pestaña ⚙ en la barra superior. Aquí se crean y editan los paneles, se gestiona la identificación de empresa y se guardan configuraciones.
  3. Añadir un panel — Pulsa ＋ Añadir panel. Asígnale nombre, tipo (Gráfica, Tabla o Tabla Dinámica), la hoja del Excel y la fila de cabecera. Pulsa ▶ Aplicar cambios para ver el resultado en el dashboard.
  4. Ver el dashboard — Pestaña 🏠 en la barra superior. Puedes cambiar entre Dashboard y Configuración en cualquier momento sin perder el estado.
  5. Ayuda — Pestaña ❓ en la barra superior para abrir este panel de instrucciones. También con Ctrl+/.
  6. Tema visual — Botón ☀️/🌙 en la barra superior para alternar entre tema oscuro y claro.
📊
Panel tipo Gráfica
Barras, líneas, circular, radar, dispersión…
▶
  1. Tipo de gráfico — 9 tipos disponibles: Barras, H.Barras (horizontal), Líneas, Área, Circular, Anillo, Radar, Polar y Dispersión.
  2. Eje X (categorías) — Selecciona la columna que define las etiquetas del eje horizontal. Puede ser texto, fechas o números.
  3. Eje Y (valores) — Marca una o varias columnas numéricas. Para cada una elige la agregación: Σ Suma, avg Media, nº Contar, min o max. Si marcas una columna de fecha como eje Y puedes agrupar por Día, Semana, Mes, Trimestre o Año.
  4. Paleta de colores — 5 paletas: Cosmic (azules/morados), Ocean (cian), Sunset (rojos/naranjas), Forest (verdes), Mono (grises).
  5. Leyenda — Activa o desactiva con el checkbox Mostrar leyenda.
📋
Panel tipo Tabla
Datos tabulares paginados con formato automático
▶
  1. Configuración básica — Selecciona la hoja del Excel y la fila de cabecera (por defecto 1). Las columnas numéricas se alinean a la derecha y los valores negativos aparecen en rojo.
  2. Columnas visibles — Si no seleccionas ninguna columna se muestran todas. Si marcas columnas en Columna etiqueta y Columnas de valores, el panel muestra solo esas columnas con los datos agregados según los filtros activos.
  3. Paginación — Campo Filas por página (por defecto 20). Navega con los botones ‹ y ›. El pie muestra el total de filas y la página actual.
  4. Modo ampliado — Al pulsar ⤢ Ampliar, la tabla muestra hasta 200 filas sin paginar con scroll vertical. Al cerrar el modo ampliado se restaura la paginación normal.
  5. Ordenar — Haz clic en cualquier cabecera de columna para ordenar ascendente/descendente.
⊞
Panel tipo Tabla Dinámica
Cruces, agrupaciones, campos calculados
▶
  1. Zonas de campos — Asigna columnas a cada zona haciendo clic en el botón de zona junto a cada campo: Filas (categorías en vertical), Columnas (cruce en horizontal) y Valores (métricas numéricas).
  2. Funciones de valor — Por cada métrica en Valores elige: Suma, Media, Mínimo, Máximo o Contar.
  3. Formato de celda — Por cada métrica: Auto, Entero, 2 dec, %, % dec, €, k (miles) o M (millones).
  4. Agrupación temporal — Los campos de fecha en Filas o Columnas permiten agrupar por: Sin agrupar, Día, Semana, Mes, Trimestre o Año.
  5. Campos calculados — Añade métricas derivadas con fórmulas entre columnas. Ejemplo: [Ventas] - [Costes].
  6. Filtros fijos — Filtros dentro del panel que se aplican antes de la agregación, independientes de los filtros del dashboard.
  7. Ordenar y Top N — Ordena por etiqueta o por métrica (asc/desc). Mostrar top N limita los resultados a los N primeros registros.
  8. Vista Tabla / Gráfica — Alterna entre tabla (con mapa de calor opcional 🌡) y gráfica (Barras, Líneas, Área, Circular…).
  9. Mapa de calor — Checkbox 🌡 Mapa calor en la vista tabla: colorea las celdas por magnitud de menor a mayor.
  10. Exportar CSV — Botón dentro del panel: descarga los datos del pivot con separador ; y codificación UTF-8, compatible con Excel en español.
📐
Distribución del dashboard
Filas, columnas y reordenación de paneles
▶
  1. Sistema de filas — El dashboard se organiza en filas independientes. Cada fila tiene su propia distribución de columnas: 1, 1+1, 1+1+1, x4, 2+1, 1+2, 3+1, 1+3, 1+2+1, 2+1+1 y 1+1+2. Puedes mezclarlas libremente.
  2. Controles de fila — Al pasar el ratón sobre una fila aparece una barra de controles: selector visual de distribución, botones ↑ / ↓ para reordenar y ✕ para eliminar la fila.
  3. Añadir fila — Botón ＋ Añadir fila al pie del dashboard (debajo de la última fila de paneles).
  4. Badge de posición — Número a la derecha del título de cada panel. Pulsa para abrir el modal de movimiento y colocarlo en cualquier otra posición disponible.
  5. Arrastrar paneles — Arrastra cualquier panel por el icono ⠿ de su cabecera. Si el destino ya tiene panel, se intercambian automáticamente.
🔍
Filtros interactivos
Globales, por panel y de fechas
▶
  1. Filtros globales — Barra colapsable sobre el dashboard. Aparece cuando todos los paneles usan la misma hoja. Filtra todos los paneles a la vez. Pulsa en la barra para expandir/colapsar los controles.
  2. Filtros por panel — Aparecen bajo la cabecera de cada panel cuando los paneles usan hojas distintas. Son independientes y se combinan con los globales (AND). También se colapsan/expanden.
  3. Filtros categóricos — Desplegables multi-selección para columnas de texto. Un badge numérico en el botón indica cuántos valores están activos.
  4. Filtros de fecha — Selectores Desde / Hasta que aparecen automáticamente en columnas de tipo fecha. Detecta formatos DD/MM/YYYY, YYYY-MM-DD y fechas de Excel.
  5. Filtros fijos (pivot) — Solo en tablas dinámicas. Se configuran dentro del panel y se aplican antes de la agregación, independientemente del resto de filtros.
  6. AutoFiltros de Excel — Si el fichero cargado tiene AutoFiltros nativos, la app los detecta y muestra un aviso en la configuración del panel.

Acciones de panel
Ampliar, exportar y consultar con IA
▶

Cada panel muestra en su cabecera, de izquierda a derecha:

  1. ⠿ Arrastrar — Icono de rejilla en el extremo izquierdo. Mantén pulsado y arrastra para reubicar el panel. También puedes arrastrar desde cualquier zona vacía de la cabecera.
  2. Título del panel — Nombre asignado en la configuración.
  3. Badge de posición — Número de orden del panel (a la derecha del título). Pulsa para moverlo a otra posición mediante el modal.
  4. 🤖 Consulta IA — Abre el drawer lateral de IA con los datos del panel cargados automáticamente.
  5. 📤 Exportar panel — Menú desplegable con: 🖼 Imagen PNG, 🖨 Imprimir, 📄 PDF, 📝 Word y 📊 Excel (.xlsx).
  6. ⤢ Ampliar — El panel ocupa toda la pantalla (modo fullscreen). Las tablas muestran hasta 200 filas sin paginar. Pulsa de nuevo para restaurar.
📤
Exportar el dashboard completo
PDF, PNG, Word, Excel e imprimir
▶
  1. 🖼 Imagen PNG — Captura el dashboard completo con fondo blanco.
  2. 🖨 Imprimir — Abre el diálogo de impresión del navegador con estilos de impresión optimizados.
  3. 📄 PDF — Genera un PDF A4 con un panel por página, incluyendo nombre de empresa y fecha.
  4. 📝 Word — Exporta el dashboard a un documento .docx con tablas de datos y cabecera de empresa.
  5. 📊 Excel (.xlsx) — Exporta todos los paneles a un libro Excel, una hoja por panel.
💡 Accede al menú de exportación global desde el botón 📤 de la barra superior.
💾
Guardar configuraciones
Guardar, cargar, exportar e importar diseños
▶
  1. 🏢 Identificación de empresa — Campo en la sección Identificación de Configuración. El nombre aparece en la cabecera de todos los documentos exportados (PDF, Word, Excel, impresiones). Se guarda junto con el diseño del dashboard.
  2. Guardar — Escribe un nombre en el campo Nombre y pulsa 💾 Guardar actual. Se guardan: paneles, distribución de filas, filtros activos, consultas rápidas, nombre de empresa y ruta del fichero.
  3. Cargar — Haz clic en el botón Cargar junto a la configuración deseada en la lista. Restaura todos los ajustes automáticamente.
  4. Exportar — Descarga el diseño como fichero .json que incluye todos los ajustes excepto las API Keys de IA.
  5. Importar — Carga un fichero .json exportado previamente y lo añade a la lista de configuraciones guardadas. También admite ficheros .conf de versiones anteriores.
  6. Autoguardado — La configuración activa se guarda automáticamente en el navegador (localStorage) cada vez que realizas cambios en paneles, filtros o distribución.
🔄
Actualizar datos del Excel
Recargar sin reconfigurar los paneles
▶
  1. Carga automática — Al abrir el HTML, el último fichero Excel se carga instantáneamente desde la caché del navegador. No necesitas volver a seleccionarlo ni dar ningún permiso. Los datos se actualizan cada vez que cargas un fichero nuevo.
  2. Recarga manual — Pulsa 📂 Cargar datos y selecciona el mismo fichero con datos actualizados. Los paneles y configuraciones se mantienen intactos y la caché se actualiza.
  3. Recarga rápida — En Chrome/Edge, si la caché no está disponible, aparece un banner amarillo. Pulsa Enter o haz clic para recargar el último fichero.
  4. Olvidar fichero — Usa 🗑 Limpiar datos o 🗑 Borrar todo en Configuración → Acciones para eliminar la caché y que no se cargue automáticamente.
  5. Atajos de teclado — Ctrl+K (o Cmd+K) para abrir el selector de fichero en cualquier momento.
🤖
Consulta IA — Uso
Analizar datos con inteligencia artificial
▶
  1. Abrir el drawer — Pulsa el icono 🤖 en la cabecera de cualquier panel. Los datos del panel se cargan automáticamente como contexto. El drawer se abre a la derecha sin tapar el dashboard.
  2. Escribir y enviar — Escribe tu pregunta en el campo de texto inferior. Enter envía el mensaje; Shift+Enter inserta una nueva línea.
  3. ⚡ Consultas rápidas — A la izquierda del campo de texto aparece un botón ⚡ (solo si hay consultas guardadas). Al pulsarlo se despliega un popup con la lista. Haz clic en una consulta para enviarla automáticamente. Cada consulta tiene un botón ✕ para eliminarla.
  4. Guardar consulta rápida — Debajo de cada mensaje tuyo aparece el botón ⚡ Guardar como consulta rápida. Las consultas se guardan en el navegador y se conservan entre sesiones. También puedes añadirlas desde ⚙ Configuración → Consultas rápidas.
  5. Contexto enviado — El panel colapsable 📋 Contexto enviado a la IA muestra las tablas incluidas, una previsualización de los datos y las instrucciones personalizadas que se envían con cada mensaje.
  6. Añadir más datos — Botón ＋ Añadir tabla dentro del drawer para incluir datos de otros paneles. La IA recibe todos los contextos en el mismo chat.
  7. Redimensionar — Arrastra el borde izquierdo del drawer para ajustar su anchura. El tamaño se mantiene durante la sesión.
  8. Instrucciones personalizadas — En ⚙ Configuración → Consulta IA, campo Instrucciones. Define el contexto de negocio, el significado de columnas, etc.
  9. Exportar conversación — Botón 📤 en la barra del drawer: 🖼 PNG, 🖨 Imprimir, 📄 PDF, 📝 Word y 📊 Excel. El nombre de empresa aparece en todos los formatos.
  10. Historial por panel — El historial de chat se mantiene por panel durante toda la sesión. Al cambiar de panel y volver, la conversación sigue intacta.
💡 Ejemplo de instrucciones útiles:
"Somos una empresa de distribución de alimentación. La columna 'Cod_Art' es el código de artículo. 'Imp_Neto' es el importe sin IVA en euros. Responde siempre en español y resume los importes en miles (k€)."

Configurar IAs disponibles
Añadir, editar y gestionar modelos de IA
▶

La aplicación soporta múltiples proveedores de IA. Ve a ⚙ Configuración → Consulta IA para gestionarlas.

Campos del formulario:
  • Nombre — Etiqueta que aparece en el selector (ej: "GPT-4o Empresa")
  • Tipo — Protocolo de la API: Claude, OpenAI compatible, Google Gemini u Ollama local
  • Endpoint — URL base de la API (ver ejemplos abajo)
  • Modelo — ID exacto del modelo a usar
  • API Key — Clave de autenticación (se guarda únicamente en el navegador, no en el fichero de diseño)
💾 Exportar / Importar configuración IA:
  • Exportar lista — Genera un fichero .json con todas las IAs configuradas, la API Key y las instrucciones del sistema. Útil para hacer copia de seguridad o compartir la configuración entre dispositivos.
  • Importar lista — Carga ese fichero y restaura automáticamente las IAs, la API Key y las instrucciones. Muestra un diálogo de confirmación indicando exactamente qué se va a restaurar.
  • Compatible con el formato antiguo (solo array de IAs) y con el nuevo formato con envoltorio.

💻 Ollama (IA local, sin coste ni API Key)

🖥 LM Studio (IA local con interfaz)

Tipo: OpenAI compatible Endpoint: http://localhost:1234/v1/chat/completions Modelo: El que cargues en LM Studio API Key: No necesaria (o cualquier texto) Instalar: lmstudio.ai → activa el servidor local en Developer tab
💡 Consejo: Puedes tener varias IAs configuradas y cambiar entre ellas desde el selector en Configuración → Consulta IA. El botón 🧪 Probar verifica que la API Key es válida antes de usarla.
🔒 Privacidad: Las API Keys se guardan únicamente en tu navegador (localStorage) y no se incluyen en el fichero de diseño del dashboard. Sí se incluyen al usar Exportar lista de IAs — guarda ese fichero en un lugar seguro. Los datos de tu Excel se envían directamente al proveedor de IA que elijas, sin pasar por ningún servidor intermedio.
